# Risk Pilot 📊

A **Position Sizing Calculator** for traders. Enter capital, risk %, entry price, and stop loss to calculate optimal lot size and view real-time crypto prices.

## Features

- 📈 Position sizing & risk calculations
- 🥧 Interactive pie chart visualization
- 💰 Live crypto prices (27+ coins via CoinGecko)

## Quick Start

1. Open `index.html` in browser
2. Enter trading parameters (capital, risk %, entry, stop loss)
3. View calculations & crypto prices

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- CoinGecko API for live prices
- CSS conic-gradient for pie chart
